# PORTFOLIO BLOG

## Description

This is a portfolio blog application built with [Nest](https://github.com/nestjs/nest) framework for my [Portfolio](https://ricqcodes.dev).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
